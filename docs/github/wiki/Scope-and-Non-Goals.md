# 범위 및 비목표

이 문서는 현재 공개 가능한 MVP 범위와 의도적으로 열지 않은 영역을 구분합니다.

## 현재 범위 (In Scope)

- 한국 노동법 검색과 법령 근거가 함께 제시되는 답변
- 로그인 없이 사용할 수 있는 AI 법률 상담과 문서 초안 흐름
- 서버 인증이 확인된 계약서 검토 -> 상담 연결 -> 답변 흐름
- 로그인 후 사용하는 기능을 위한 Firebase Google Sign-In
- 사건 기록 보관함과 기록 삭제
- 안정적인 데모 확인을 위한 예시 사례 흐름
- 화면에서 생성하는 사업장 변경 사유 정리서 예시 초안
- 로컬 실행과 데모 검증 가이드
- 개발 환경 우선 클라우드 전환 문서

## 비목표 (Non-Goals)

- 최종 법률 판단 또는 변호사 대체
- 장기 운영용 법률 상담 서비스라는 주장
- 계약서 검토 기반 서버 초안 생성
- 계약서 검토 기반 초안 전용 로그인 API
- 독립 상담 연결 전용 화면
- Recovery 흐름
- 추가 문서/상담 시나리오 확장
- 완전 삭제, 파일 물리 삭제, 계정 삭제, 보관 기간 정책
- 브라우저 저장소 또는 공개 문서에 사건 원문 데이터 노출
- 공개 미러 저장소에서 배포 수행
- 공모전 데모 상태를 장기 운영 서비스로 표현

## 경계가 필요한 이유

프로젝트는 데모 안정성, 법령 근거 확인, 개인정보 최소화를 우선합니다. 공개 API
계약을 바꾸거나 보관 정책을 확장하거나 새 접근 권한 검토가 필요한 기능은 현재
MVP 범위 밖에 둡니다.

## 시나리오 경계

AI 법률 상담은 로그인 없이 확인할 수 있는 대표 데모입니다. 계약서 검토 연결은
로그인 사용자를 위한 흐름입니다. 계약서 검토 결과는 사용자의 이전 사건 설명을
AI 법률 상담 질문에 연결하지만, 사건 연결과 참고용 정보일 뿐 법적 근거가 아닙니다.
인용 조문을 새로 만들거나 검색된 법령 근거를 대체하지 않습니다.
