# 프로젝트 수행 및 완성

법대로(LawMainRoad)는 법령 검색, 근거 기반 답변, 문서 초안, 사용자별 기록
연결을 하나의 노동권 보호 MVP로 묶은 프로젝트입니다.

## 목표

외국인 근로자가 근로계약, 기숙사, 임금체불, 부당해고, 사업장 변경 같은
상황을 한국어 노동법 근거와 함께 정리하고, 다음 행동을 준비할 수 있게
돕는 것이 목표입니다.

## 현재 완료 범위

기준일: `2026-05-13`

| 트랙 | 결과 |
|---|---|
| 법령 검색 기반 | 법령 조각 코퍼스, 검색, 법령 근거가 함께 제시되는 답변 구현 |
| AI 법률 상담 | 로그인 없이 질문 -> 답변 -> 문서 정보 입력 -> 초안 흐름 구현 |
| 계약서 검토 연결 | 계약서 검토 기록, 상담 연결, 이어지는 답변 구현 |
| 로그인 | Firebase Google Sign-In과 서버 인증 확인 구현 |
| 사건 기록 | 기록 보관함, 저장된 사건 선택, 기록 삭제 구현 |
| 데모 안정성 | 예시 사례 흐름과 사전 점검 스크립트 구현 |
| 클라우드 전환 | 개발 환경 기본 확인과 `www.law-main-road.cloud` 공개 도메인 연결 완료 |
| 공개 문서 | README, Wiki, YouTube 시연 영상/MP4 백업 연결 정리 |

## 수행 요약

법대로(LawMainRoad)는 먼저 법령 코퍼스와 검색 기반 답변을 안정화한 뒤,
AI 법률 상담과 문서 초안 흐름을 대표 데모로 정리했습니다. 이후 계약서 검토는
Firebase Google Sign-In과 서버 인증 확인을 붙여 계약서 검토 기록, 상담 연결,
이어지는 상담 답변, 사건 기록 보관함, 기록 삭제까지 확장했습니다.

마지막 화면 정리는 서버, API, 데이터 스키마 변경 없이 메인, 계약서 검토,
AI 법률 상담, 사건 기록 화면의 대비, 카드 구조, 안내 문구, 진행 상태, 주의 문구를
다듬은 작업입니다.

## 주요 사용자 가치

- 법률 질문을 단순 LLM 답변이 아니라 검색된 법령 근거와 함께 확인할 수 있습니다.
- 사용자가 입력하지 않은 사실은 단정하지 않고 추가 확인 항목으로 남깁니다.
- 계약서 검토에서 확인한 쟁점을 AI 법률 상담 질문에 이어갈 수 있습니다.
- 답변 근거가 충분할 때 지원되는 문서 초안까지 이어집니다.

## 검증 스냅샷

현재 프로젝트는 다음 수준의 검증을 기준으로 문서화합니다.

- 백엔드 import 확인
- 문서 초안 검증 스크립트
- 프론트엔드 빌드
- 데모 사전 점검 스크립트
- 주요 데모 흐름의 수동 브라우저 확인
- 공개 검토용 YouTube 시연 영상과 MP4 백업 파일 확인

전체 검색/답변 평가는 검색, 답변 생성, 임베딩, 데이터베이스 내용, 공개 API 계약이
바뀐 경우에만 실행합니다.

## 남은 작업

남은 작업은 기능 확장보다 데모 기간 모니터링, 수동 화면 QA, 공모전 데모 운영 상태
검토에 초점을 둡니다.

- 수동 화면 QA와 인쇄 미리보기 근거 정리
- 공개 도메인 데모 기간 모니터링과 공모전 데모 운영 상태 검토
- 향후 장기 운영 서비스 전환을 결정하는 경우 별도 운영 검토
